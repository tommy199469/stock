1. install python
2. install python library -- boto3 , pandas , beautifulSoup4 , openpyxl
3. create s3_config.py file.
4. put your config into s3_config.py
    AWS_ACCESS_KEY_ID = 'XXXXXX'
    AWS_SECRET_ACCESS_KEY = 'XXXXX'
    AWS_DEFAULT_REGION = "XXXXX"
    BUCKET_NAME = "XXXXX"
    
5. run main.py
